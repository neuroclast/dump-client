About this Site

While this site is designed to be fully functional, its primary goal is as a technology demonstrator and for learning and advancing various frameworks.

This site is built on AWS Elastic Beanstalk using Spring Boot, Angular 5, Bootstrap 4, Prism, and various other supporting modules and libraries.

As the primary goal of this site is neither UI or workflow design, various parts of those elements have been made to mimic highly functional sites, primarily pastebin.net.
